# CSSTree CLI

## How to use:

```bash
> npm i -g csstree-cli
> csstree /path/to/style.css
> echo ".hello { world: 123 }" | csstree
```

```
> csstree -h
Usage:

  csstree [filename] [options]

Options:

  -c, --compact            Compact output
  -h, --help               Output usage information
  -v, --version            Output version
```
